* **Tambah Tugas Baru:** Masukkan deskripsi tugas dan tanggal jatuh tempo.
* **Manajemen Status:** Tandai tugas sebagai **Completed** (Selesai) atau **Pending** (Tertunda) dengan *checkbox*.
* **Statistik Real-time:** Menampilkan total tugas, tugas selesai, tugas tertunda, dan persentase progres secara langsung.
* **Pencarian & Filter:**
    * Fungsi pencarian *real-time* berdasarkan kata kunci.
    * Tombol Filter untuk beralih tampilan antara **Semua Tugas**, **Selesai**, dan **Tertunda**.
* **Sorting:** Mengurutkan daftar berdasarkan **Nama Tugas** atau **Tanggal Jatuh Tempo**.
* **Penyimpanan Data Persisten:** Semua tugas disimpan di **Local Storage** *browser* pengguna, sehingga data tidak hilang saat halaman di-*refresh* atau ditutup.
